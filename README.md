README intro
SAFE - Women Safety AI is a mobile-style safety web application designed to support women in emergency situations. The app listens for distress speech after the SOS button is pressed, converts voice into live transcript, detects high-risk phrases, calculates a hybrid danger score, captures the user’s location, and sends an SMS alert with a Google Maps link through a Flask and Twilio backend.

Key features

Animated SOS emergency button
Live speech-to-text using Web Speech API
Distress keyword detection and highlighting
Repeated phrase detection
Hybrid danger score with AI-style analysis
Live location tracking with embedded map
Automatic danger-state UI
Flask backend for emergency alert API
Twilio SMS integration for emergency contact notification
Tech stack

Frontend: HTML, CSS, JavaScript
APIs: Web Speech API, Geolocation API
Backend: Python Flask
Messaging: Twilio API
